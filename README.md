# Nutrients_Calc
## 概要 
PFCそれぞれの栄養素について計算をするアプリです

## クラスごとの説明
### Main.java
- 単純に計算メソッドを呼び出し、結果を出力します

### Calorie.java
- PFCそれぞれのカロリー数をenumで定義をしています

### NutrientCalculation.java
- カロリー変換、計算メソッドです
- 総カロリー量を計算するメソッドです
- 摂取量のバリデーションチェックをしています

### DecimalUtils.java
- 四捨五入の計算は共通化の名目でUtilsクラスに記載しました
- 今回は引数をMain.javaから渡していますが、本来はアプリの入力から値を受け取るので。。
  - 数値かどうかを判断するメソッドをUtilsクラスに実装しました
  - 摂取量(g)は基本正の数値なので、負の数や文字列等は受け付けないべきです
  - 入力されたことを想定して、テストケースに確認すべきケースを実装しました
